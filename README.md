# BlueSaver
Projeto de MC426.  
Desenvolvido em Flask com apresentação das funcionalidades em um Swagger.

## Para rodar o servidor FLASK, pode utilizar:
### I - Bash com seleção:
```bash
bash run.sh
```
### OU

### II - Flask run:
```bash
python3 -m flask --app app run --debug
```

## Para visualizar as rotas, acesse:
```
localhost:5000
```
